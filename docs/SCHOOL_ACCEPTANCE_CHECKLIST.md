# School acceptance checklist

School: ____________________  Academic year: __________  Pilot stream: __________

## Academic configuration

- [ ] Headteacher/DOS verified school, EMIS and UNEB centre details.
- [ ] All offered subjects, class levels, streams and teacher allocations are correct.
- [ ] Authorised NCDC topics, outcomes, Activities of Integration and rubrics are imported.
- [ ] S1–S4 and S5–S6 assessment policies were approved by the school.
- [ ] Every S5/S6 subject plan has ongoing evidence components and one Term III annual summative component.
- [ ] Student enrolments, subject registrations and guardian links were reconciled against source registers.

## End-to-end acceptance scenarios

- [ ] Teacher creates and HOD approves a scheme of work and lesson plan.
- [ ] Teacher records observation, conversation and product evidence for a learner.
- [ ] A project milestone, portfolio item, reflection and rubric rating are completed.
- [ ] Missing evidence remains missing—not zero—and blocks report publication.
- [ ] HOD and DOS moderation locks approved evidence; authorised reopening is audited.
- [ ] A learner is promoted and the previous-year report still shows the correct historical stream and subjects.
- [ ] Parent and learner can see only their published reports and verified portfolio evidence.
- [ ] S5/S6 processing rejects termly summative tests and incomplete annual components.
- [ ] UNEB preflight blocks missing subjects, CA, S3 project scores and pending approvals.
- [ ] PDF reports, class exports and transcripts reconcile with manually calculated samples.

## Operational acceptance

- [ ] `check --deploy`, `check_school_readiness` and the full test suite pass.
- [ ] HTTPS, secure cookies, role permissions, audit logs and session expiry were tested.
- [ ] Daily database/media backups and off-site replication are scheduled.
- [ ] A restoration drill succeeded and recovery time was recorded.
- [ ] Staff training, support ownership, incident response and data-retention procedures are documented.
- [ ] Existing official records will run in parallel for the agreed pilot term.
